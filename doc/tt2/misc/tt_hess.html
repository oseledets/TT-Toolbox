tt_hess

PURPOSE

Numerical Hessian in a given point

SYNOPSIS

function [Hess]=tt_hess(tt, indext, h)

DESCRIPTION 

Numerical Hessian in a given point
   [HESS]=TT_HESS(TT, INDEXT, H) Computes the numerical Hessian of the
   TT-tensor TT computed at INDEXT. The grid size is supposed to be equal
   to H
