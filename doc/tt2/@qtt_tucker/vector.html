vector

PURPOSE

Transforms a QTT-Tucker matrix into QTT-Tucker vector

SYNOPSIS

function [qt]=vector(qt)

DESCRIPTION 

Transforms a QTT-Tucker matrix into QTT-Tucker vector
   [QT]=VECTOR(QT) It just makes each factor of the Tucker decomposition a
   TT-tensor instead of the TT-matrix
