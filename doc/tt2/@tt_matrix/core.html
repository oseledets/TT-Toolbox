core

PURPOSE

Converts TT-matrix to TT1 cell-array format

SYNOPSIS

function [tt] = core(tt1,varargin)

DESCRIPTION 

Converts TT-matrix to TT1 cell-array format
   [TT]=CORE(TT1) Converts TT-matrix to TT1 cell array format

   [TT]=CORE(TT1,NUM) Extracts the NUM-th core of the TT-matrix
