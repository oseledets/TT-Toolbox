subsref

PURPOSE

Evaluate cores of TT-matrix and fields of the TT-matrix structure

SYNOPSIS

function elem = subsref(tt,s)

DESCRIPTION 

 Evaluate cores of TT-matrix and fields of the TT-matrix structure
   A=TT{I} computes the I-th core of the TT-representation
