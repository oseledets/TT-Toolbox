horzcat

PURPOSE

C=[A,B]

SYNOPSIS

function [c]=horzcat(a,b)

DESCRIPTION 

C=[A,B]
   C=HORZCAT(A,B) Horizontal cat of two block-row TT-tensors
   The result has r(d+1) = sum(r(d+1,A)+r(d+1,B)). The first ranks 
   should be equal for A and B
