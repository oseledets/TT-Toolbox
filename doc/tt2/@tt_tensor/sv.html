sv

PURPOSE

Computes singular values of all unfoldings of a TT-tensor

SYNOPSIS

function [sval]=sv(tt)

DESCRIPTION 

Computes singular values of all unfoldings of a TT-tensor
   [SVAL]=SV(TT) Computes all nonzero singular values of all unfoldings 
    and returns result as a cell array
