col

PURPOSE

Computes the columns of a block TT-tensor

SYNOPSIS

function [y]=col(x, k, varargin)

DESCRIPTION 

Computes the columns of a block TT-tensor 
   [Y]=COL(X,K)
   It is useful for block algorithms when r(d+1) is not equal to 1
   For analogous thing on r(1) see ROW(X,K)
