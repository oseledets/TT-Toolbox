core

PURPOSE

Converts TT-tensor TT1 to old-cell array format.

SYNOPSIS

function [tt] = core(tt1,varargin)

DESCRIPTION 

Converts TT-tensor TT1 to old-cell array format.
   [TT]=CORE(TT1) transforms to the TT1.0 format

   [TT]=CORE(TT1,K) extracts the k-th core
