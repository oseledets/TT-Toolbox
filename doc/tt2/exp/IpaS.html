IpaS

PURPOSE

A special bidiagonal matrix in the QTT-format

SYNOPSIS

function [M]=IpaS(d, a)

DESCRIPTION 

A special bidiagonal matrix in the QTT-format
   [M]=IPAS(D, A)
   Generates I+a*S_{-1} matrix in the QTT-format:
   1 0 0 0
   a 1 0 0
   0 a 1 0
   0 0 a 1
 Convenient for Crank-Nicolson and time gradient matrices
